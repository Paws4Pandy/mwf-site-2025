About Liquid Glass

The Liquid Glass effect creates Apple's signature glass morphism with realistic refraction, distortion, and light interaction. Perfect for creating premium, modern UI elements that stand out.

  • Multiple refraction modes: standard, polar, prominent, shader
  • Realistic light distortion and glass-like appearance
  • Customizable intensity, scale, and displacement
  • Performance optimized with WebGL shaders
  • Perfect for buttons, cards, and interactive elements

Implementation Guide

To use the Liquid Glass effect in your mortgage website:

  • Import the LiquidGlass component from the cloned repository
  • Wrap your UI elements with the component
  • Choose the appropriate refraction mode for your design
  • Adjust intensity and scale for the perfect balance
  • Use on buttons, cards, modals, and interactive elements
